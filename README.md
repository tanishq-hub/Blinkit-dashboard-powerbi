# Blinkit-dashboard-powerbi

Project Overview

This project showcases an interactive Power BI dashboard developed to analyze Blinkit sales data. The dashboard provides a comprehensive view of sales performance, item categories, outlet characteristics, customer ratings, and historical trends to support data-driven decision-making.

Objectives
Analyze total and average sales performance
Understand sales distribution across item types and fat content
Compare outlet performance by size, location, and tier
Identify trends in outlet establishment over time
Evaluate customer ratings and item visibility

Key Metrics
Total Sales: ₹1.20M
Average Sales: ₹140.99
Item Count: 9K
Average Rating: 3.92

Dashboard Features
Sales trend analysis based on outlet establishment year
Item-wise sales comparison across product categories
Fat content distribution analysis
Outlet performance comparison by size and location tier
Outlet type analysis including supermarkets and grocery stores
Interactive filters for item type, outlet size, and location

Tools and Technologies
Power BI for data modeling and visualization
DAX for calculated measures and KPIs
Standard visualization techniques such as bar charts, line charts, donut charts, and tables

Key Insights
Tier 3 outlets contribute the highest overall sales
Medium-sized outlets generate the largest share of revenue
Fruits, snacks, and household items are the top-selling categories
Sales peaked notably around 2018
Customer ratings remain consistently close to 4 across outlet types

Conclusion
The Blinkit Sales Analysis Dashboard effectively converts raw sales data into actionable insights. It enables stakeholders to evaluate performance, identify trends, and make informed business decisions using interactive and visually clear reports.
